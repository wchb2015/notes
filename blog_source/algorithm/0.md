### 几种常用排序算法的比较

| 算法 | 平均时间复杂度 | 原地排序 | 额外空间 | 稳定排序 |
| :-----: |:-------:| :-----:| :-----:|:-----:|
| Selection Sort | O(n^2) | 是  | O(1)  | 否   | 
| Insertion Sort | O(n^2) | 是 |O(1) | 是 | 
| Merge Sort     | O(nlogn) | 否 |O(n) | 是 |
| Quick Sort     | O(nlogn) | 是 |O(logn) | 否 |
| Heap Sort      | O(nlogn) | 是 |O(1) | 否 |
